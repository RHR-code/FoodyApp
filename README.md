# FoodyApp

FoodyApp is the number one place to find all your favorite foods.You can not only buy foods but also sell yours foods to our large audience.

[Live Website Link](https://foody-app-theta.vercel.app/)

### Features

- User can add Foods
- User can buy Foods
- User can view their Foods
- User can delete their foods
